Module: SpiSeq::Internal::Utils::ModuleFunctionForwardable

Defined in:
lib/spiseq/internal/utils.rb

Overview

A simplified version of Forwardable that defines module_functions rather than plain methods. Extend a module with this and call def_mod_func_delegators to define module_functions that delegate to the given methods on the object. This is not as flexible as Forwardable! Namely it assumes that this module is extending another module, and that the object to which you are forwarding is the name of a constant.
