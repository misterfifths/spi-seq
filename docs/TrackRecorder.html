Records incoming MIDI notes and creates a Track by quantizing those notes to a grid.

Recording is stopped and started by a MIDI CC (the cc argument). The value of the CC message is ignored. Any notes that are still on when recording is stopped by the CC are treated as if they ended at the same time that the CC was received.

bpm is the beats per minute at which the track is to be played back, and granularity is the slot granularity for the resulting track. Together these two determine the duration of a slot in the track, and thus control how to map between wall-clock seconds and steps in the track. A shorter granularity - or a faster BPM - means that the timing of incoming MIDI notes can be represented more precisely, which may or may not be desirable.

By default, the track will last for the duration that recording was enabled via the CC; if there was silence at the beginning or end of recording, there will be rests on either end of the resulting track. If true, trim_start and trim_end will remove those rests from the corresponding ends of the track.

If for any reason the resulting track would be empty (e.g. if no MIDI notes were observed while recording), this function returns nil.

min_gate specifies the minimum gate for single notes in the track; if a recorded MIDI event would translate to a single step with a gate less than that value, its duration is rounded up to min_gate instead. Also, if an event translates to a sequence of tied steps followed by one step with a partial gate, that final step will be dropped if its gate is less than min_gate.

Parameters:

  • cc (Integer) —

    The MIDI CC number that will begin and end recording.

  • cc_port (String, nil) (defaults to: nil) —

    The MIDI port to monitor for CC messages. If nil, falls back to the global default set with #use_cc_control_defaults or all ports (i.e. "*") if no default was set.

  • cc_channel (Integer, String, nil) (defaults to: nil) —

    The MIDI channel to monitor for CC messages. If nil, falls back in the same manner as cc_port.

  • port (String, nil) (defaults to: nil) —

    The MIDI port to monitor for notes. If nil, falls back to the global default set with Sonic Pi's use_midi_defaults, or all ports (i.e. "*") if no default was set.

  • channel (Integer, String, nil) (defaults to: nil) —

    The MIDI channel to monitor for notes. If nil, falls back in the same manner as port.

  • bpm (Integer, nil) (defaults to: nil) —

    The BPM to use when mapping between real-world seconds and slots in the resulting track. If nil, uses the current Sonic Pi BPM.

  • granularity (Symbol, Number, NoteLength) (defaults to: NoteLength::Eighth) —

    The granularity of the resulting track. Can be a NoteLength or a value understood by NoteLength.new.

  • trim_start (Boolean) (defaults to: false) —

    Whether to remove rests from the beginning of the returned track.

  • trim_end (Boolean) (defaults to: false) —

    Whether to remove rests from the end of the returned track.

  • min_gate (Number) (defaults to: 0.1) —

    The minimum gate for a Step in the track. See above for details.

  • quantize_gates (Boolean) (defaults to: true) —

    If true, gates will be snapped upwards to the nearest multiple of min_gate.

  • ignore_vel (Boolean) (defaults to: false) —

    If true, the velocity of incoming MIDI notes will be ignored and all steps in the track will have the default velocity of 127.

Returns:

  • (Track, nil)

def self.record(cc:, cc_port: nil, cc_channel: nil,
                port: nil, channel: nil,
                bpm: nil, granularity: NoteLength::Eighth,
                trim_start: false, trim_end: false,
                min_gate: 0.1, quantize_gates: true,
                ignore_vel: false)
  cc_port, cc_channel = __resolve_cc_port_and_channel(cc_port, cc_channel)
  port, channel = __resolve_midi_port_and_channel(port, channel)

  start_time = end_time = timeline = nil
  ExtApi.with_real_time do
    start_time, end_time, timeline = record_timeline(control_cc: cc, cc_port: cc_port, cc_channel: cc_channel,
                                                     port: port, channel: channel)
  end

  start_time = nil if trim_start
  end_time = nil if trim_end

  timeline_to_track(timeline,
                    start_time: start_time, end_time: end_time,
                    bpm: bpm, granularity: granularity,
                    min_gate: min_gate, quantize_gates: quantize_gates,
                    ignore_vel: ignore_vel)
end
